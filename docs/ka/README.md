 ### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљрЃљ README-рЃўрЃА рЃљрЃЏ рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃерЃў?
 рЃЮрЃарЃўрЃњрЃўрЃюрЃљрЃџрЃўрЃА рЃћрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃў [README](https://github.com/cobbzilla/mobiletto/blob/master/README.md)
 рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃ«рЃљрЃарЃЋрЃћрЃќрЃћрЃЉрЃў -- *рЃерЃћрЃАрЃгрЃЮрЃарЃћрЃЉрЃћрЃЉрЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃўрЃАрЃљрЃАрЃљрЃџрЃЏрЃћрЃЉрЃћрЃџрЃўрЃљ!* рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃњрЃљрЃЏрЃЮрЃљрЃњрЃќрЃљрЃЋрЃюрЃЮрЃЌ [рЃњрЃљрЃДрЃЋрЃљрЃюрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ GitHub-рЃќрЃћ](https://github.com/cobbzilla/mobiletto/pulls),
 рЃљрЃю рЃЌрЃБ рЃљрЃа рЃњрЃАрЃўрЃљрЃЏрЃЮрЃЋрЃюрЃћрЃЉрЃЌ рЃљрЃЏрЃўрЃА рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃљ, [рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃў](https://github.com/cobbzilla/mobiletto/issues)

 рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџ GitHub рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃА рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ, рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮрЃЌ:
 * рЃЕрЃљрЃарЃЌрЃћрЃЌ рЃњрЃЋрЃћрЃарЃЊрЃўрЃА URL (рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ/рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃА рЃќрЃЮрЃџрЃўрЃЊрЃљрЃю)
 * рЃерЃћрЃўрЃбрЃљрЃюрЃћрЃЌ рЃќрЃБрЃАрЃбрЃў рЃбрЃћрЃЦрЃАрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ (рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ/рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃЊрЃљрЃю)
 * рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ рЃљрЃдрЃгрЃћрЃарЃЮрЃЌ рЃарЃљ рЃљрЃарЃўрЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў -- рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃў? рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ рЃњрЃљрЃбрЃћрЃ«рЃўрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃЏрЃћ?
 * рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃерЃћрЃАрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЌ рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃўрЃА рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃљ, рЃљрЃю рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ
 * **р▓њрЃЏрЃљрЃЊрЃџрЃЮрЃЉрЃЌ!**

 ## рЃарЃљрЃбрЃЮрЃЏ Mobiletto?

 ### рЃюрЃљрЃ«рЃЋрЃљрЃЏрЃЊрЃўрЃА рЃњрЃљрЃЏрЃДрЃўрЃЊрЃЋрЃћрЃџрЃА рЃЕрЃљрЃЎрЃћрЃбрЃЋрЃљ!
 рЃдрЃарЃБрЃЉрЃџрЃЮрЃЋрЃљрЃюрЃў рЃАрЃљрЃфрЃљрЃЋрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃърЃарЃЮрЃЋрЃљрЃўрЃЊрЃћрЃарЃћрЃЉрЃА рЃљрЃЦрЃЋрЃЌ рЃерЃћрЃБрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃћрЃџрЃў API. рЃЏрЃљрЃЌрЃљрЃф рЃЎрЃў, рЃЋрЃўрЃюрЃф рЃўрЃЉрЃарЃФрЃЋрЃўрЃА "S3 рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃЎрЃћрЃю"
 рЃљрЃЦрЃЋрЃЌ рЃўрЃЊрЃўрЃЮрЃАрЃўрЃюрЃЎрЃарЃљрЃбрЃБрЃџрЃў рЃЦрЃфрЃћрЃЋрЃћрЃЉрЃў.

 рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃўрЃарЃЕрЃћрЃЋрЃЌ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃњрЃљрЃЏрЃДрЃўрЃЊрЃЋрЃћрЃџрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃърЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЌрЃБ рЃЎрЃЮрЃЊрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЏрЃљрЃЌ API-рЃерЃў, рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃљрЃърЃерЃў рЃљрЃгрЃЋрЃЊрЃўрЃЌ
 рЃљрЃ«рЃџрЃљ рЃљрЃЏ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў. рЃарЃљрЃф рЃЊрЃарЃЮ рЃњрЃљрЃЊрЃўрЃА рЃЊрЃљ рЃЎрЃЮрЃЊрЃў рЃњрЃарЃЮрЃЋрЃЊрЃћрЃЉрЃљ, рЃЏрЃЮрЃЋрЃљрЃГрЃарЃћрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ рЃ«рЃЊрЃћрЃЉрЃљ
 рЃАрЃБрЃџ рЃБрЃцрЃарЃЮ рЃњрЃљрЃБрЃАрЃљрЃФрЃџрЃўрЃАрЃў. рЃЎрЃћрЃЌрЃўрЃџрЃў рЃўрЃДрЃЮрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃЮрЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃДрЃўрЃЊрЃЋрЃћрЃџрЃўрЃА рЃЕрЃљрЃЎрЃћрЃбрЃЋрЃўрЃА рЃЏрЃ«рЃўрЃљрЃарЃБрЃџ рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃерЃў!

 Mobiletto рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃљрЃЏ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃГрЃарЃћрЃџрЃљрЃЊ. рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃўрЃЌ mobiletto-рЃА API-рЃќрЃћ, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ
 рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃЋрЃљрЃўрЃЊрЃћрЃарЃћрЃЉрЃў рЃЊрЃљ рЃўрЃфрЃЮрЃЊрЃћрЃЌ, рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃцрЃћрЃюрЃљ рЃўрЃЊрЃћрЃюрЃбрЃБрЃарЃљрЃЊ рЃўрЃЦрЃфрЃћрЃЋрЃљ.

 ### рЃЋрЃарЃфрЃћрЃџрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ
 рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃФрЃдрЃЮрЃџрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃўрЃЊрЃћрЃюрЃбрЃБрЃарЃў рЃЦрЃфрЃћрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА 60+ рЃбрЃћрЃАрЃбрЃўрЃЌ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃФрЃдрЃЮрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
 рЃЕрЃЋрЃћрЃю рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃА рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃўрЃЌ:
 * рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃљ: рЃЕрЃљрЃарЃЌрЃБрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃў
 * Redis cache: рЃЕрЃљрЃарЃЌрЃБрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃў

 рЃћрЃА рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃЏрЃерЃЋрЃўрЃЊрЃљрЃЊ, рЃарЃЮрЃЏ mobiletto рЃўрЃЦрЃфрЃћрЃЋрЃљ рЃўрЃњрЃўрЃЋрЃћ, рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџ рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ,
 рЃЊрЃљ рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃЕрЃљрЃарЃЌрЃљрЃЋрЃЌ рЃЌрЃБ рЃљрЃарЃљ рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃљрЃА рЃЊрЃљ/рЃљрЃю рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃљрЃА.

 ### рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ
 рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ Mobiletto рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃћрЃЉрЃў:
 * `s3` : Amazon S3
 * `b2` : Backblaze B2
 * `local` : рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃў рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

 *рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃгрЃЋрЃџрЃўрЃџрЃў рЃдрЃарЃБрЃЉрЃџрЃЮрЃЋрЃљрЃюрЃў рЃАрЃљрЃфрЃљрЃЋрЃўрЃА рЃЏрЃћрЃбрЃў рЃърЃарЃЮрЃЋрЃљрЃўрЃЊрЃћрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃАрЃљрЃГрЃћрЃарЃљрЃЊ рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃўрЃАрЃљрЃАрЃљрЃџрЃЏрЃћрЃЉрЃћрЃџрЃўрЃљ!*

 ## рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃЊрЃљ рЃЊрЃљрЃцрЃўрЃюрЃљрЃюрЃАрЃћрЃЉрЃљ
 рЃЏрЃћ рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉ рЃЋрЃўрЃДрЃЮ рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃў рЃдрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃА рЃерЃћрЃЏрЃЦрЃЏрЃюрЃћрЃџрЃў. рЃЏрЃћ рЃЋрЃЏрЃБрЃерЃљрЃЮрЃЉрЃЊрЃў
 рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃА рЃўрЃюрЃЊрЃБрЃАрЃбрЃарЃўрЃљрЃерЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃгрЃћрЃџрЃўрЃљ, рЃЊрЃљрЃЋрЃўрЃгрЃДрЃћ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃЋрЃДрЃўрЃЊрЃћ рЃўрЃАрЃўрЃюрЃў рЃАрЃљрЃ»рЃљрЃарЃЮ рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃћрЃЉрЃерЃў.
 рЃфрЃЮрЃбрЃљ рЃ«рЃюрЃўрЃА рЃгрЃўрЃю рЃАрЃљрЃЏрЃАрЃљрЃ«рЃБрЃарЃў рЃЊрЃљрЃЋрЃЎрЃљрЃарЃњрЃћ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃљрЃа рЃЏрЃљрЃЦрЃЋрЃА

 рЃљрЃАрЃћ рЃарЃЮрЃЏ, рЃЏрЃћ рЃЋрЃљрЃърЃўрЃарЃћрЃЉ рЃЊрЃљрЃЋрЃгрЃћрЃарЃЮ рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃљ рЃЊрЃљ рЃЋрЃюрЃљрЃ«рЃЮ, рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ

 рЃЌрЃБ рЃњрЃАрЃўрЃљрЃЏрЃЮрЃЋрЃюрЃћрЃЉрЃЌ рЃљрЃЏ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃЊрЃџрЃЮрЃЉрЃћрЃџрЃў рЃЋрЃўрЃЦрЃюрЃћрЃЉрЃў рЃЌрЃБрЃюрЃЊрЃљрЃф рЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА

 ## рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ
 рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ `npm` рЃљрЃю `yarn` . рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃџрЃЉрЃљрЃЌ рЃњрЃАрЃБрЃарЃЌ `lite` рЃЋрЃћрЃарЃАрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃа рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃДрЃЋрЃћрЃџрЃљ
 рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃџрЃў README рЃцрЃљрЃўрЃџрЃћрЃЉрЃў:

    npm install mobiletto-lite
    yarn add mobiletto-lite

 рЃЌрЃБ рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃњрЃАрЃБрЃарЃЌ README рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃДрЃЋрЃћрЃџрЃљ рЃћрЃюрЃљрЃќрЃћ, рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ рЃАрЃарЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ:

    npm install mobiletto
    yarn add mobiletto

 ## р▓АрЃгрЃарЃљрЃцрЃў рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ
 рЃЏрЃЮрЃЎрЃџрЃћ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў mobiletto `s3` рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

 рЃћрЃА рЃЎрЃЮрЃЊрЃў рЃћрЃарЃЌрЃюрЃљрЃўрЃарЃљрЃЊ рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃЊрЃљ, рЃЌрЃБ рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў рЃўрЃЦрЃюрЃћрЃЉрЃЮрЃЊрЃљ `b2` рЃљрЃю `local` .

    const storage = require('mobiletto')
    const bucket = await storage.connect('s3', aws_key, aws_secret, {bucket: 'bk'})

    // list objects: returns array of metadata objects
    const listing = await bucket.list()
    const dirList = await bucket.list('some/dir/')
    const everything = await bucket.list('', {recursive: true})

    // write an entire file
    let bytesWritten = await bucket.writeFile('some/path', someBufferOfData)

    // write a file from a stream/generator
    bytesWritten = await bucket.write('some/path', streamOrGenerator)

    // read an entire file
    // returns null if an exception would otherwise be thrown
    const bufferOrNull = await bucket.safeReadFile('some/path')

    // stream-read a file, passing data to callback
    const bytesRead = await bucket.read('some/path', (chunk) => { ...do something with chunk... } )

    // remove a file, returns the path removed
    let removed = await bucket.remove('some/path') // removed is a string

    // remove a directory, returns array of paths removed
    removed = await bucket.remove('some/directory', {recursive: true}) // removed is now an array!

 ## рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
 рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃБрЃцрЃарЃЮ рЃЋрЃарЃфрЃћрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃЋрЃўрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃерЃћрЃЏрЃЮрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃБрЃЏрЃћрЃбрЃћрЃАрЃЮрЃЉрЃљрЃА:

    const { mobiletto } = require('mobiletto')

    // General usage
    const api = await mobiletto(driverName, key, secret, opts)

    // To use 'local' driver:
    // * key: base directory
    // * secret: ignored, can be null
    // * opts object:
    // * readOnly: optional, never change anything on the filesystem; default is false
    // * fileMode: optional, permissions used when creating new files, default is 0600. can be string or integer
    // * dirMode: optional, permissions used when creating new directories, default is 0700. can be string or integer
    const local = await mobiletto('local', '/home/ubuntu/tmp', null, {fileMode: 0o0600, dirMode: '0700'})

    // To use 's3' driver:
    // * key: AWS Access Key ID
    // * secret: AWS Secret Key
    // * opts object:
    // * readOnly: optional, never change anything on the bucket; default is false
    // * bucket: required, name of the S3 bucket
    // * region: optional, the AWS region to communicate with, default is us-east-1
    // * prefix: optional, all read/writes within the S3 bucket will be under this prefix
    // * delimiter: optional, directory delimiter, default is '/' (note: always '/' when encryption is enabled)
    const s3 = await mobiletto('s3', aws_key, aws_secret, {bucket: 'bk', region: 'us-east-1'})

    // To use 'b2' driver:
    // * key: Backblaze Key ID
    // * secret: Backblaze Application Key
    // * opts object:
    // * readOnly: optional, never change anything on the bucket; default is false
    // * bucket: required, the ID (**not the name**) of the B2 bucket
    // * prefix: optional, all read/writes within the B2 bucket will be under this prefix
    // * delimiter: optional, directory delimiter, default is '/' (note: always '/' when encryption is enabled)
    // * partSize: optional, large files will be split into chunks of this size when uploading
    const b3 = await mobiletto('b2', b2_key_id, b2_app_key, {bucket: 'bk', partSize: 10000000})

    // List files
    api.list() // --> returns an array of metadata objects

    // List files recursively
    api.list({ recursive: true })

    // List files in a directory
    const path = 'some/path'
    api.list(path)
    api.list(path, { recursive: true }) // also supports recursive flag

    // Visit files in a directory -- visitor function must be async
    api.list(path, { visitor: myAsyncFunc })
    api.list(path, { visitor: myAsyncFunc, recursive: true })

    // The `list` method throws MobilettoNotFoundError if the path does not exist
    // When you call `safeList` on a non-existent path, it returns an empty array
    api.safeList('/path/that/does/not/exist') // returns []
    
    // Read metadata for a file
    api.metadata(path) // returns metadata object

    // The `metadata` method throws MobilettoNotFoundError if the path does not exist
    // When you call `safeMetadata` on a non-existent path, it returns null
    api.safeMetadata('/tmp/does_not_exist') // returns null
    
    // Read a file
    // Provide a callback that writes the data someplace
    const callback = (chunk) => { ... write chunk somewhere ... }
    api.read(path, callback) // returns count of bytes read

    // Read an entire file at once
    const data = await api.readFile(path) // returns a byte Buffer of the file contents
    
    // Read an entire file at once
    // returns null if an exception would otherwise be thrown
    const bufferOrNull = await bucket.safeReadFile('some/path')

    // Write a file
    // Provide a generator function that yields chunks of data
    const generator = function* () {
      while ( ... more-data-to-return ... ) {
        data = ... load-data ...
        yield data
      }
    }
    local.api(path, generator) // returns count of bytes written

    // Write an entire file at once (convenience method)
    await api.writeFile(path, bufferOrString) // returns count of bytes written

    // Delete a file
    // Quiet param is optional (default false), when set errors will not be thrown if the path does not exist
    // Always returns a value or throws an error.
    // Return value may be a single string of the file removed, or an array of all files removed (driver-dependent)
    const quiet = true
    api.remove(path, {quiet}) // returns single path removed

    // Recursively delete a directory and do it quietly (do not report errors)
    const recursive = true
    const quiet = true
    api.remove(path, {recursive, quiet}) // returns array of paths removed

 ## рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
 `metadata` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃћрЃарЃЌрЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.
 рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃљрЃЊ, рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ `list` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃЊрЃљрЃю рЃљрЃарЃўрЃА рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃАрЃўрЃЋрЃў.

 рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ:

    {
      "name": "fully/qualified/path/to/file",
      "type": "entry-type",
      "size": size-in-bytes,
      "ctime": creation-time-epoch-millis,
      "mtime": modification-time-epoch-millis
    }

 `type` рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА `file` , `dir` , `link` , рЃљрЃю `special` .

 рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃбрЃўрЃърЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ, `list` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃа рЃЊрЃљрЃљрЃЉрЃарЃБрЃюрЃЮрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЋрЃћрЃџрЃў. `name` name" рЃЊрЃљ `type` рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў
 рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў. рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃў `metadata` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃЊрЃљрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏ рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃА.

 ## рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃА рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџрЃў рЃАрЃбрЃўрЃџрЃў
 рЃЏрЃЮрЃљрЃ«рЃЊрЃўрЃюрЃћ рЃАрЃарЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ `connect` :

    const storage = require('mobiletto')
    const opts = {bucket: 'bk', region: 'us-east-1'}
    const s3 = await storage.connect('s3', aws_key, aws_secret, opts)
    const objectData = await s3.readFile('some/path')

 ## рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃљ
 Mobiletto рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА <a href="https://redis.io">redis</a> рЃЦрЃћрЃерЃўрЃЌ.

 Mobiletto рЃерЃћрЃћрЃфрЃЊрЃћрЃЉрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљрЃА redis рЃћрЃњрЃќрЃћрЃЏрЃърЃџрЃљрЃарЃЌрЃљрЃю 127.0.0.1:6379

 рЃЌрЃЦрЃЋрЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃБрЃњрЃБрЃџрЃћрЃЉрЃћрЃџрЃДрЃЮрЃЌ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЏрЃљрЃЌрЃњрЃљрЃюрЃў:
 * рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `MOBILETTO_REDIS_HOST` env var, mobiletto рЃЊрЃљрЃАрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃџрЃљрЃЊ рЃљрЃЦ рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў рЃ░рЃЮрЃАрЃбрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ
 * рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `MOBILETTO_REDIS_PORT` env var, рЃћрЃА рЃърЃЮрЃарЃбрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў

 Mobiletto рЃерЃћрЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЌрЃљрЃЋрЃўрЃА рЃарЃћрЃЊрЃўрЃА рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃА рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃўрЃЌ `_mobiletto__` . рЃЌрЃЦрЃЋрЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮрЃЌ рЃћрЃА
 `MOBILETTO_REDIS_PREFIX` env var-рЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

 рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃљрЃДрЃћрЃюрЃЮрЃЌ рЃЌрЃўрЃЌрЃЮ рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃљ `opts.redisConfig` рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃЌ:

    const redisConfig = {
        enabled: true, // optional, default is true. if false other props are ignored
        host: '127.0.0.1',
        port: 6379,
        prefix: '_mobiletto__'
    }
    const opts = { redisConfig, bucket: 'bk', region: 'us-east-1' }
    const s3 = await storage.connect('s3', aws_key, aws_secret, opts)

 ### рЃљрЃа рЃњрЃАрЃБрЃарЃЌ рЃ«рЃћрЃџрЃљрЃ«рЃљрЃџрЃў рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃљ?
 рЃњрЃљрЃЏрЃЮрЃАрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ: рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЊрЃљрЃЏрЃДрЃљрЃарЃћрЃЉрЃўрЃАрЃљрЃА `enabled: false` рЃЌрЃЦрЃЋрЃћрЃюрЃА `opts.redisConfig` рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃерЃў.

 рЃарЃЮрЃњрЃЮрЃарЃф рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃњрЃљрЃюрЃўрЃ«рЃўрЃџрЃћрЃЉрЃљ, рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃарЃЌрЃЋрЃљ рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃљрЃЊ рЃўрЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљрЃќрЃћ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃћрЃЋрЃА рЃЏрЃћрЃб рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃА
 рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ.

 ### рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃћрЃЉрЃў
 **рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў рЃАрЃљрЃфрЃљрЃЋрЃў**: рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃљ/рЃгрЃћрЃарЃљ рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃќрЃћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЮрЃЊрЃюрЃљрЃЋ рЃюрЃћрЃџрЃўрЃљ,
 рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃўрЃарЃњрЃЋрЃџрЃўрЃЋ рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ (рЃарЃљрЃАрЃљрЃф рЃќрЃЮрЃњрЃўрЃћрЃарЃЌрЃў рЃарЃљрЃЏ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА) рЃАрЃљрЃЎрЃЏрЃљрЃЮрЃЊ рЃФрЃЋрЃўрЃарЃўрЃљ. рЃарЃћрЃЊрЃўрЃА рЃЦрЃћрЃерЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
 рЃЏрЃЮрЃњрЃфрЃћрЃЏрЃЌ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃю рЃќрЃарЃЊрЃљрЃА.

 рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў рЃЦрЃћрЃерЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЎрЃљрЃарЃњрЃљрЃЊ, рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃљрЃЦрЃЋрЃЌ рЃЉрЃћрЃЋрЃарЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃА/рЃгрЃљрЃерЃџрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў.
 рЃЕрЃљрЃгрЃћрЃарЃўрЃА рЃљрЃю рЃгрЃљрЃерЃџрЃўрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃљрЃарЃдрЃЋрЃћрЃЋрЃА рЃЏрЃЌрЃћрЃџ рЃЦрЃћрЃерЃА, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃў рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљрЃА
 рЃБрЃљрЃ«рЃџрЃћрЃАрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў.

 ### CLI рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў
 рЃЌрЃБ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ CLI рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ [mobiletto-cli](https://www.npmjs.com/package/mobiletto-cli),
 рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃАрЃБрЃарЃЌ рЃЕрЃљрЃарЃЌрЃЮрЃЌ redis рЃЦрЃћрЃерЃў, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃА рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ `mo` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃЌ.

 `mirror` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊ рЃљрЃАрЃџрЃА рЃћрЃарЃЌрЃў рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃўрЃЊрЃљрЃю рЃЏрЃћрЃЮрЃарЃћрЃќрЃћ.
 рЃўрЃА рЃљрЃа рЃљрЃгрЃљрЃарЃЏрЃЮрЃћрЃЉрЃА рЃарЃљрЃўрЃЏрЃћ рЃърЃарЃЮрЃфрЃћрЃАрЃА рЃАрЃљрЃарЃЎрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃџрЃљрЃЊ рЃЊрЃарЃЮрЃЌрЃљ рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў. рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ `mirror` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ
 рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЊрЃљрЃЎрЃљрЃарЃњрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.

 рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ `mirror` рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃЏрЃарЃўрЃфрЃ«рЃЋрЃћрЃџрЃћрЃЉрЃўрЃЌ, рЃЌрЃБ рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃцрЃљрЃўрЃџрЃў рЃўрЃДрЃЮ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ
 рЃАрЃљрЃарЃЎрЃўрЃАрЃћрЃЉрЃБрЃџрЃў рЃЊрЃљ рЃарЃљрЃЏрЃЊрЃћрЃю рЃцрЃљрЃўрЃџрЃА рЃ░рЃЦрЃЮрЃюрЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў:

    {
      success: count-of-files-mirrored,
      errors: count-of-files-with-errors
    }

 рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЊрЃўрЃЊрЃў рЃюрЃљрЃЎрЃарЃћрЃЉрЃўрЃА рЃљрЃАрЃљрЃ«рЃЋрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃерЃарЃЮрЃЏрЃљрЃбрЃћрЃЋрЃљрЃЊрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃбрЃљрЃарЃБрЃюрЃљрЃарЃўрЃљрЃюрЃЮрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃюрЃАрЃўрЃБрЃарЃў

 рЃАрЃћрЃЏрЃљрЃюрЃбрЃўрЃЎрЃўрЃА `mirror` , рЃќрЃЮрЃњрЃ»рЃћрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃЏрЃљрЃЉрЃюрЃћрЃЋрЃћрЃџрЃў рЃўрЃДрЃЮрЃА рЃўрЃЏрЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃљ, рЃЌрЃБ рЃЋрЃўрЃю рЃљрЃарЃўрЃА рЃўрЃА
 рЃЏрЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃџрЃў рЃЊрЃљ рЃЋрЃўрЃю рЃљрЃарЃўрЃА рЃЏрЃгрЃћрЃарЃљрЃџрЃў. рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃњрЃЮрЃарЃф рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ: "рЃЏрЃљрЃарЃфрЃ«рЃћрЃюрЃљ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃўрЃА"
 рЃљрЃарЃўрЃА рЃюрЃўрЃЋрЃЌрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃћрЃюрЃўрЃГрЃћрЃЉрЃљ (рЃљрЃарЃћрЃЎрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ) рЃЊрЃљ РђърЃЏрЃљрЃарЃ»рЃЋрЃћрЃюрЃљ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃўрЃАРђю (
 рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃў `mirror` рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃАрЃЌрЃЋрЃўрЃА) рЃљрЃарЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ (рЃАрЃљрЃарЃЎрЃўрЃАрЃћрЃЉрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃўрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ).

 р▓арЃљ рЃ«рЃЊрЃћрЃЉрЃљ? рЃфрЃљрЃџрЃЎрЃћ "рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃАрЃЋрЃџрЃўрЃА" (dirent) рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ (рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў) рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃА рЃарЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЏрЃљрЃАрЃерЃў
 рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ (aka dirent рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ).
 * `list` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА, рЃерЃўрЃцрЃарЃљрЃЋрЃА рЃЕрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃџ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃњрЃќрЃљрЃА; рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА
 * `list` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў рЃБрЃцрЃарЃЮ рЃљрЃарЃљрЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃўрЃљ, рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃЊрЃў рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃўрЃЌ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
 * `write` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃгрЃћрЃарЃА dirent рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃерЃЮрЃЉрЃџрЃўрЃА dirent рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў, рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃљрЃЊ; рЃерЃћрЃЏрЃЊрЃћрЃњ рЃгрЃћрЃарЃА рЃцрЃљрЃўрЃџрЃА
 * `write` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃА рЃЏрЃЮрЃ░рЃДрЃЋрЃћрЃЉрЃљ O(N) рЃЕрЃљрЃгрЃћрЃарЃљ, N = рЃАрЃўрЃдрЃарЃЏрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃљрЃерЃў
 * рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ `remove` рЃљрЃерЃЮрЃарЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА dirent рЃцрЃљрЃўрЃџрЃА рЃЊрЃљ рЃЏрЃўрЃА рЃЏрЃерЃЮрЃЉрЃћрЃџрЃА, рЃЌрЃБ рЃфрЃљрЃарЃўрЃћрЃџрЃўрЃљ, рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃљрЃЊ; рЃерЃћрЃЏрЃЊрЃћрЃњ рЃгрЃљрЃерЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃА
 * рЃљрЃарЃљрЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў `remove` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃћрЃЋрЃА O(N) рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃА рЃЊрЃљ рЃърЃЮрЃбрЃћрЃюрЃфрЃўрЃБрЃарЃљрЃЊ рЃљрЃЏрЃЊрЃћрЃюрЃўрЃЋрЃћ рЃгрЃљрЃерЃџрЃљрЃА, N = рЃАрЃўрЃдрЃарЃЏрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃљрЃерЃў
 * рЃЊрЃўрЃЊ рЃЊрЃљ рЃдрЃарЃЏрЃљ рЃцрЃљрЃўрЃџрЃБрЃа рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃќрЃћ рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў `remove` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃФрЃЋрЃўрЃарЃў

 рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЏрЃ«рЃарЃўрЃЊрЃљрЃю рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃўрЃф рЃЎрЃў, рЃЏрЃЮрЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњрЃћ рЃАрЃарЃБрЃџрЃў рЃ«рЃўрЃџрЃЋрЃљрЃЊрЃЮрЃЉрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџ рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА
 рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ, рЃЌрЃБрЃюрЃЊрЃљрЃф рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, рЃЏрЃљрЃўрЃюрЃф рЃ«рЃћрЃЊрЃљрЃЋрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃА рЃЊрЃљ рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃцрЃљрЃўрЃџрЃўрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃерЃў рЃЊрЃљ рЃЌрЃљрЃю
 рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃФрЃљрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃљ, рЃљрЃдрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃўрЃА рЃќрЃЮрЃњрЃўрЃћрЃарЃЌрЃў рЃљрЃю рЃЏрЃЌрЃћрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ.
 *рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃЉрЃарЃбрЃДрЃћрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.*
 рЃЏрЃЮрЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњрЃћрЃЏ рЃљрЃа рЃўрЃфрЃЮрЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА/рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў, рЃЌрЃБ рЃЏрЃљрЃЌ рЃљрЃАрЃћрЃЋрЃћ рЃљрЃа рЃўрЃфрЃЮрЃЊрЃюрЃћрЃю рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃљ
 рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў рЃљрЃю рЃАрЃ«рЃЋрЃљрЃњрЃЋрЃљрЃарЃљрЃЊ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃљрЃарЃдрЃЋрЃўрЃљ рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃљ. рЃДрЃЋрЃћрЃџрЃљ рЃцрЃАрЃЮрЃюрЃў рЃерЃћрЃЕрЃћрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃерЃўрЃю!

 ### рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃљ
 рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃюрЃћрЃџрЃў. рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃў рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃюрЃћрЃџрЃў рЃўрЃДрЃЮрЃА.
 рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃљ redis-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃФрЃљрЃџрЃўрЃљрЃю рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЦрЃћрЃерЃў рЃЕрЃљрЃЦрЃарЃћрЃЉрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃА рЃљрЃю рЃгрЃљрЃерЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ.

 ## рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃарЃЮрЃбрЃљрЃфрЃўрЃљ
 рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃбрЃЮрЃбрЃЮ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃ«рЃљрЃџрЃў рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃЌ, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃАрЃљрЃ«рЃћрЃЌ рЃЏрЃљрЃАрЃерЃў рЃФрЃЋрЃћрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў:

    const storage = require('mobiletto')

    const oldEncryption = { key: .... }
    const oldStorage = await storage.connect('s3', aws_key, aws_secret, {bucket: 'bk', region: 'us-east-1'}, oldEncryption)

    const newEncryption = { key: .... }
    const newStorage = await storage.connect('s3', aws_key, aws_secret, {bucket: 'zz', region: 'us-east-1'}, newEncryption)

    newStorage.mirror(oldStorage) // if oldStorage is very large, this may take a looooooong time...

 ## рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў
 рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў рЃљрЃарЃўрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў JS рЃцрЃљрЃўрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА "storageClient" рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА рЃљрЃЏ рЃ«рЃћрЃџрЃЏрЃЮрЃгрЃћрЃарЃўрЃЌ:

    function storageClient (key, secret, opts)

 * `key` : рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў, рЃЌрЃЦрЃЋрЃћрЃюрЃў API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў ( `local` рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ)
 * `secret` : рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў, рЃЌрЃЦрЃЋрЃћрЃюрЃў API рЃАрЃљрЃўрЃЊрЃБрЃЏрЃџрЃЮ (рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ `local` рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў)
 * `opts` : рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў, рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў рЃљрЃарЃўрЃА рЃЌрЃўрЃЌрЃЮ рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў:
 * `local` "-рЃўрЃАрЃЌрЃЋрЃўрЃА, `fileMode` рЃЊрЃљ `dirMode` рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.
 * `s3` рЃАрЃљрЃГрЃўрЃарЃЮрЃљ `bucket` рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ. рЃљрЃарЃЕрЃћрЃЋрЃўрЃЌрЃў рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃўрЃљ:
    * `region`: the S3 region, default is us-east-1
    * `prefix`: a prefix to prepend to all S3 paths, default is the empty string
    * `delimiter`: the directory delimiter, default is '/'

 рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА storageClient рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃБрЃюрЃЊрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃЮрЃА рЃћрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:

    // Test the driver before using, ensure proper configuration
    async testConfig ()

    // List files in path (or from base-directory)
    // If recursive is true, list recursively
    // If visitor is defined, it will be an async function. await the visitor function on each file found
    // Otherwise, perform the listing and return an array of objects
    async list (path, recursive = false, visitor = null) // path may be omitted
    
    // Read metadata for a path
    async metadata (path)
    
    // Read a file
    // callback receives a chunk of data. endCallback is called at end-of-stream
    async read (path, callback, endCallback = null)

    // Write a file
    // driver must be able to handle a generator or a stream
    async write (path, generatorOrReadableStream)

    // Remove a file, or recursively delete a directory
    // returns a string of a single path removed, or an array of multiple paths removed
    async remove (path, recursive = false, quiet = false)

 ## рЃерЃћрЃАрЃЋрЃџрЃљ
 Mobiletto рЃўрЃДрЃћрЃюрЃћрЃЉрЃА [winston](https://www.npmjs.com/package/winston) рЃЪрЃБрЃарЃюрЃљрЃџрЃўрЃА рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃА.

 рЃЪрЃБрЃарЃюрЃљрЃџрЃў **рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃА рЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ **рЃљрЃарЃљрЃАрЃЊрЃарЃЮрЃА** рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃА, рЃАрЃљрЃўрЃЊрЃБрЃЏрЃџрЃЮрЃћрЃЉрЃћрЃЉрЃА,
 рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ.

 ### рЃЪрЃБрЃарЃюрЃљрЃџрЃўрЃА рЃЊрЃЮрЃюрЃћ
 рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `MOBILETTO_LOG_LEVEL` рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў рЃЪрЃБрЃарЃюрЃљрЃџрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЊрЃљрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ, рЃћрЃарЃЌрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ
 [https://www.npmjs.com/package/winston#logging-levels]-рЃерЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў `npm` (https://www.npmjs.com/package/winston#logging-levels)

 рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў рЃЊрЃЮрЃюрЃћ рЃљрЃарЃўрЃА `error` . рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃ«рЃЏрЃљрЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃћ рЃљрЃарЃўрЃА `silly` , рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃЏрЃЪрЃљрЃЏрЃљрЃЊ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃўрЃљ
 рЃљрЃа рЃерЃћрЃЊрЃўрЃА "рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃўрЃА" рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ `debug`

    MOBILETTO_LOG_LEVEL=silly # maximum logs!

 ### рЃЪрЃБрЃарЃюрЃљрЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃў
 рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃљрЃЊ, рЃџрЃЮрЃњрЃћрЃарЃў рЃгрЃћрЃарЃА рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃА. рЃЪрЃБрЃарЃюрЃљрЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃерЃў рЃњрЃљрЃАрЃљрЃњрЃќрЃљрЃЋрЃюрЃљрЃЊ рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `MOBILETTO_LOG_FILE`
 рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў. рЃцрЃљрЃўрЃџрЃерЃў рЃерЃћрЃАрЃЋрЃџрЃўрЃАрЃљрЃА рЃЪрЃБрЃарЃюрЃљрЃџрЃћрЃЉрЃў рЃљрЃдрЃљрЃа рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃќрЃћ.
